Acorn - Programming Language

One of my more ambitious project, Acorn, is a programming language made from scratch. It's the most challenging project I've taken on so far. Building a program that takes a program with infinite combinations of legal input became a mind-boggling realization. This project took a month to complete and near a year to refine. Acorn is an interpreted-programming language written in Python, which means that Acorn code becomes translated into and evaulated in Python. The goal of this project was to learn more about language creation, lexing, recursive-descent parsing, interpreting, language syntax/semantics, parse tress, abstract syntax trees, abstract-memory handling, lexical scoping, etc...

Iris - Assistive Technology

Iris is an iOS app that assist those with visual impairments. Iris uses Tesseract - OCR to analyze image-text and converts it to digital text. A feature of Iris translates further this image-text by turning it into viberational braille. See a demonstration video here.
