Acorn

One of my more ambitious project, Acorn, is a programming language made from scratch. It's the most challenging project I've taken on so far. Building a program that takes a program with infinite combinations of legal input became a mind-boggling realization. This project took a month to complete and near a year to refine. From left-to-right, you will see the order of how Acorn code becomes translated and ran. The process goes Acorn source code, lexed source, tokenized source and parse, finally Acorn output. Acorn is an interpreted-programming language written in Python, which means that Acorn code becomes translated into and evaulated in Python. The goal of this project was to learn more about language creation, lexing, recursive-descent parsing, interpreting, language syntax/semantics, parse tress, abstract syntax trees, abstract-memory handling, lexical scoping, etc...
